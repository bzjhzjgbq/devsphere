# AGENTS.md

## Project
DevSphere is a developer community website, not a personal blog, not an admin dashboard, and not an AI landing page.

## Primary goal
The current goal is to create polished, modern, production-looking frontend UI.
Do not prioritize backend logic or complex business logic unless explicitly requested.

## Visual style
Reference style:
- GitHub
- Linear
- Notion
- Vercel

UI rules:
- light neutral background
- dark text
- restrained teal accent
- clean spacing
- subtle border and shadow
- no cyberpunk
- no neon glow
- no oversized hero
- no heavy glassmorphism
- no AI-generated landing page look

## Product feel
The site should feel like a real developer community platform:
- content-first
- readable
- structured
- modern
- human-designed

## Current priorities
1. Home page visual polish
2. Project page polish
3. User center UI
4. Article page UI

## Constraints
- Keep using React + Vite + Tailwind CSS + React Router
- Use mock data unless explicitly told to connect backend
- Prefer reusable components
- Keep file structure clean
- Avoid unnecessary dependencies

## Done when
- layout is visually balanced
- pages feel product-grade
- components are reusable
- responsive behavior is acceptable
- no placeholder section looks empty or unfinished
